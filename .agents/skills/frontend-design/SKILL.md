# Frontend Design

Use this skill to design beautiful, distinctive, and modern layouts.

## Guidelines
- Avoid plain or generic system fonts (e.g. system UI / standard Inter defaults) when a premium look is needed.
- Curate harmonious, tailored color palettes (HSL-based, Sleek dark modes).
- Add subtle micro-animations (hover transitions, layout morphing) for interactivity.
- Ensure layout responsiveness across mobile, tablet, and ultra-wide screens.
