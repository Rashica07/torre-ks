# Test Driven Development

Use this skill to strictly enforce test-driven development.

## TDD Workflow
1. Write a failing test for the desired feature or bug fix.
2. Run the test suite to confirm it fails.
3. Write the minimal code necessary to make the test pass.
4. Refactor the code while keeping tests green.
