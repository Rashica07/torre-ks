# MCP Builder

Use this skill when creating or updating custom Model Context Protocol servers.

## Workflow
- Use standard MCP SDKs (e.g., Python, TypeScript).
- Design tools with clear parameter schemas and description tags.
- Secure MCP server execution by limiting tool access to safe system paths.
