# Webapp Testing

Use this skill to design, write, and run E2E browser tests.

## Rules
- Target critical user journeys (e.g., login, checkout, registration, main workflows).
- Write atomic, self-contained tests with clear assertions.
- Do not check in secrets or credentials. Use environment variables.
