# Implement Design

Use this skill when translating Figma or other mockups to code.

## Guidelines
- Match color specs, typography scales, line-heights, and spacing margins precisely.
- Ensure responsive break points are handled cleanly.
- Verify component states (default, hover, focus, disabled, active) are all implemented.
- Use semantic HTML tags.
