# Code Simplifier

Use this skill to refactor complex code, align formatting with project conventions, and prune dead paths.

## Rules
- Seek and destroy over-engineered patterns (excessive layers of abstraction, premature optimizations).
- Maintain existing behavior while reducing lines of code.
- Write simple, direct, readable logic over clever or cryptic syntax.
- Ensure all tests still pass post-refactor.
